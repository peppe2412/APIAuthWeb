# API Auth Web 

Progetto, autenticazione utenti lato web usando l'API del progetto [APIAuthLaravel](https://github.com/peppe2412/ApiAuthLaravel).

### Libreria, per le chiamate API HTTP -> [Axios](https://axios-http.com/docs/intro):
```bash
 npm install axios
```
### Libreria, per le rotte e navigation -> [React Router](https://reactrouter.com/);
```bash
npm i react-router
```
